# Glossary

| 术语 | 定义 |
|------|------|
| `source_trade_date` | 数据所基于的真实收盘交易日。由交易日历服务按市场查询前一交易日得到。用户主界面统一展示此日期。 |
| `computed_at` | 离线批处理计算完成的时间戳。通常为交易日次日凌晨（如 02:00）。仅供内部运维和排障使用，禁止作为用户口径日期。 |
| `snapshot_date` | 快照落盘写入时间。不等于数据归属的收盘日期。与 `source_trade_date` 语义不同。 |
| 批次级元数据 | 同一批离线计算产出的所有结果共享的元信息，包括 `source_trade_date`、`computed_at` 等。 |
| 交易日历服务 | 按市场（A 股/港股）维护的交易日历，支持查询前一交易日。新模块接入 `source_trade_date` 时必须依赖此服务。 |
| 日级收盘后产出 | 任何基于当日收盘数据、在次日凌晨离线计算产出的策略、榜单、象限、组合等数据。 |
