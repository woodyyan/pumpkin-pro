# 资金星图需求

## 背景

独立项目 `liquidity-layer-monitor` 已有单页「资金星系」，用东方财富公开行情接口展示 A 股高流动性样本的估值、成交额、板块资金流和 PoC 估值锚。该能力迁入卧龙网站后命名为「资金星图」。

## 目标

- 路由为 `/capital-map`。
- 主站导航放入「看板」分组，位于「市场全景」之后。
- 首期仅支持 A 股。
- 页面公开可见，与「市场行情」「市场全景」权限口径一致。
- 保持原网站刷新频率、算法和功能不变。
- UI 适配卧龙浅色/深色主题。

## 功能需求

1. 展示 A 股 PE×成交额散点星图。
2. 展示核心统计：全市场股票数、样本成交额、上涨/下跌数量、PoC 估值锚。
3. 展示板块成交额占比与主力净流强度。
4. 展示 PE 5 倍分箱成交额分布，并标出 PoC 分箱。
5. 展示成交额 Top 个股与主力净流入 Top 板块。
6. 前端每 60 秒自动刷新，并提供手动刷新按钮。
7. 外部数据刷新失败时，后端可返回最近成功快照并标记 stale。
8. 页面底部必须展示公开数据源和投资风险口径说明。

## 非目标

- 不支持港股。
- 不新增登录权限。
- 不建设历史快照回放。
- 不在首期建设 admin 运维面板。
- 不承诺东方财富公开接口为正式商用授权数据源。
