# Changelog: 卧龙 AI 精选模拟组合指标改造

## 2026-06-02

- **体验优化**: 模拟组合顶部指标改为“累计收益、成立天数、昨日收益率、本月收益率、最大回撤、波动率、日胜率”，更方便用户判断收益区间与持有体验。
- **体验优化**: 当前成分股新增“买入价、最新价、较买入价收益”，避免把收盘后静态数据误读成盘中“今天涨跌”。
- **技术调整**: 模拟组合收益图和后端结果彻底去 benchmark 化，只保留组合自身收益曲线与风险指标。

## 2026-06-14

- **体验优化**: `portfolio-tracking` 页面取消市场 / 组合双 tab，改为 A股、港股双分区下各 2 张组合卡片平铺，首屏即可同时对比 4 个组合。
- **体验优化**: A股区使用暖色系、港股区使用冷色系 section 容器与 badge，强化市场边界。
- **体验优化**: 单卡信息密度重排为“核心指标 + 次级指标 + 迷你收益曲线 + 精简成分股 + 折叠调仓”，兼顾总览效率与细节可读性；后续移除了公告条与顶部辅助说明，保留更轻的页面文案。
- **测试调整**: 组件源码契约测试从“tab 切换”改为“市场分区平铺”，同时校验市场视觉系统、四卡结构，以及公告与辅助说明文案的删除状态。


## 2026-07-01

- **运维优化**: 新口径模拟组合管理新增事实表完整度展示，区分“仅有初始化资金”“可同步”“缺开盘价”等状态，并给出下一步操作建议。
- **可观测性增强**: `同步最新事实表` 返回每个组合的同步摘要，显示锚点、最新信号、生成估值日数量和说明，避免同步无结果时无法判断原因。
- **流程澄清**: Admin 文案明确“补齐建仓开盘价”只补前置价格，不会生成持仓或调仓；正确顺序为补价 → 同步事实表 → 验证一致性。
